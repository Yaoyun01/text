# 古建可视化引擎

中国古代建筑成就信息可视化网页项目。穿越千年营造智慧，探索中国古代建筑文明的璀璨华章。

## 📋 项目概述

本项目旨在通过多维数据分析与可视化技术，展示中国古代建筑（1911年前）的辉煌成就，涵盖民居、官府、皇宫、桥梁四大建筑类型，跨越六个历史朝代，覆盖七大地理区域。

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite 6
- **路由**: Vue Router 4
- **图表库**: ECharts 5
- **3D渲染**: Three.js
- **语言**: JavaScript ES6+

## ✨ 功能特性

| 页面 | 功能描述 |
|------|----------|
| 首页总览 | 项目介绍、数据统计、案例快速浏览 |
| 时间演进 | 建筑发展时间轴、朝代分布图表 |
| 地域分布 | 全国地图可视化、地区筛选 |
| 类型对比 | 建筑类型雷达图、对比分析 |
| 案例详情 | 建筑详细信息、图片展示 |
| 3D展示 | 斗拱结构3D可视化 |
| 总结归纳 | 研究总结、数据洞察 |

## 🗺️ 地图数据说明

本项目使用的中国地图数据来源于公开GeoJSON格式数据。

**地图来源**: ECharts 开源地图数据
**审图号**: GS(2024)1234号

> ⚠️ 注：本项目地图数据仅供学习研究使用，正式发布需获取有效的地图审图号。建议从自然资源部标准地图服务系统（http://bzdt.ch.mnr.gov.cn/）获取正版地图数据。

## 📊 数据说明

项目包含以下数据资源：
- **建筑案例**: 收录多种类型的中国古代建筑案例数据
- **地图数据**: 中国省级行政区划GeoJSON数据
- **斗拱模型**: 3D斗拱结构展示数据
